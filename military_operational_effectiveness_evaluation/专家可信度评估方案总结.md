# 专家可信度评估与权重修正方案完整文档

## 一、方案概述

本方案实现了基于专家可信度的AHP权重修正系统，核心目标是通过评估专家的主观可信度、客观可信度和把握度，对专家给出的指标权重进行科学修正。

## 三、核心评估流程

### 步骤1：加载专家背景数据
```python
experts_background = load_experts_background()
```
- 从 `experts` 表读取10位专家的基本信息
- 包含：资历年限、职称、学历、工作年限

### 步骤2：加载专家AHP权重数据
```python
experts_ahp = load_expert_ahp_weights(batch_id='AHP-2026-002')
```
- 加载20个 `_weight` 字段（专家对各指标的权重打分）
- 加载20个 `_confidence` 字段（专家对各指标的把握度，0-1范围）
- 构建 `weight_matrix` (10×20) 和 `confidence_matrix` (10×20)

### 步骤3：计算主观可信度
基于专家背景的量化评分：

**影响力得分（δe）**：
- 院士/将军：100分
- 专家：80-90分
- 高工/副教授：70-80分
- 其他：60-70分

**专业知识得分（δk）**：
- 博士 + 高职称 + 长工作年限：90-100分
- 硕士 + 中级职称：70-85分
- 本科：60-75分

**主观可信度计算**：
```
Ωi = 0.4 × δe + 0.6 × δk
```
范围：0-100分

### 步骤4：一致性检验
对20个指标分别计算变异系数（CV）：

**单指标CV计算**：
```
CV_j = (标准差 / 平均值) × 100%
```

**一致性等级**：
- CV < 15%：高度一致（优秀）
- 15% ≤ CV ≤ 25%：较为一致（良好）
- 25% < CV ≤ 35%：轻度分歧（一般）
- CV > 35%：严重分歧（不可接受）

**异常值处理**：
- 对于CV > 20%的指标，删除最远离平均值的专家评分
- 重新计算CV，观察改善效果

### 步骤5：计算专家离散度（客观指标）

**核心公式（正确版）**：
```
对每位专家i：
  1. 计算每个指标j的平均值：μ_j = mean(所有专家对指标j的评分)
  2. 计算该专家对各指标的偏差：d_ij = w_ij - μ_j
  3. 计算偏差的标准差：σ_d = std(d_i1, d_i2, ..., d_i20)
  4. 计算整体平均值：μ_overall = mean(所有专家所有指标评分)
  5. 专家CV = (σ_d / μ_overall) × 100%
```

**意义**：
- 测量专家评分相对于群体平均的偏离程度
- CV越低，说明该专家与群体意见越一致

### 步骤6：计算客观可信度
采用**分段线性衰减函数**：

```python
if CV <= 15%:
    β = 1.0                          # 完全可信
elif CV <= 25%:
    β = 1.0 - (CV - 15) / 10 * 0.5  # 从1.0衰减到0.5
elif CV <= 35%:
    β = 0.5 - (CV - 25) / 10 * 0.5  # 从0.5衰减到0.0
else:
    β = 0.0                          # 不可信
```

**示例**：
- CV = 12%：β = 1.000
- CV = 20%：β = 0.750
- CV = 30%：β = 0.250
- CV = 40%：β = 0.000

### 步骤7：计算综合可信度
```
综合可信度 = 0.4 × (主观可信度/100) + 0.6 × 客观可信度
```

**权重分配**：
- 40% 主观（专家背景）
- 60% 客观（评分一致性）

### 步骤8：计算专家最终可信度（公式4-2）

**关键步骤**：
1. 计算每位专家的平均把握度：
   ```
   α_i = mean(α_i1, α_i2, ..., α_i20)
   ```

2. 计算专家最终可信度：
   ```
   λ_i = 0.5 × 综合可信度 + 0.5 × 平均把握度
   ```

**实际数据示例**：
| 专家 | 综合可信度 | 平均把握度 | 最终可信度 |
|------|-----------|----------|----------|
| 张军 | 0.9378 | 0.7395 | **0.8387** |
| 刘雪梅 | 0.9097 | 0.7490 | **0.8294** |
| 郑雪 | 0.7760 | 0.8505 | **0.8133** |
| 刘芳 | 0.7675 | 0.7255 | **0.7465** |

**特点**：
- 郑雪虽然综合可信度不高（0.776），但把握度最高（0.851），最终可信度排第4
- 整合了专家的客观评价和主观自信心

### 步骤9：权重修正（公式4-15）

**修正公式（正确版）**：
```
对每个二级指标j：
  ω_j = Σ(λ_i × w_ij)
  
其中：
  - λ_i：专家i的最终可信度（已包含把握度信息）
  - w_ij：专家i对指标j的原始权重
```

**重要说明**：
- ❌ **错误做法**：`ω_j = Σ(λ_i × α_ij × w_ij)` （重复使用把握度）
- ✅ **正确做法**：`ω_j = Σ(λ_i × w_ij)` （λ已包含把握度）

### 步骤10：两级归一化

**第一步：一级指标内归一化**
```python
for 每个一级指标k:
    该类别的二级指标索引: indices_k
    类别内总权重: sum_k = Σ(ω_j), j ∈ indices_k
    
    for 每个二级指标j in indices_k:
        归一化权重_j = (ω_j / sum_k) × 一级指标权重_k
```

**第二步：全局验证**
```python
所有20个二级指标权重之和 = 1.000000
```

**实际结果**：
- 修正权重总和：1.000000 ✅
- 原始权重总和：1.000000 ✅

## 七、关键技术要点

### 7.1 专家CV的正确计算方法

**❌ 错误方法**（原始）：
```python
专家CV = std(专家自己的20个评分) / mean(专家自己的20个评分)
```
**问题**：只测量专家内部评分波动，无法反映与群体的差异

**✅ 正确方法**（当前）：
```python
# 1. 计算每个指标的群体平均值
indicator_means = mean(所有专家对各指标的评分, axis=0)

# 2. 计算该专家相对于群体的偏差
deviations = 专家评分 - indicator_means

# 3. 计算偏差的标准差
std_of_deviations = std(deviations)

# 4. 计算整体平均值
overall_mean = mean(所有评分)

# 5. 专家CV
专家CV = (std_of_deviations / overall_mean) × 100%
```

### 7.2 把握度的使用逻辑

**关键原则**：把握度只在计算专家最终可信度时使用一次

```
步骤1：λ_i = 0.5 × 综合可信度 + 0.5 × 平均把握度
      ↑ 在这里整合把握度

步骤2：ω_j = Σ(λ_i × w_ij)
      ↑ 不再重复使用指标级把握度α_ij
```

**原因**：
- λ_i 已经包含了专家的整体自信心水平
- 如果再乘以α_ij，会导致把握度被平方（过度惩罚）
- 指标级把握度α_ij可以在后续具体应用中单独使用

### 7.3 归一化的两级处理

**为什么需要两级归一化？**

1. **一级指标内归一化**：保持各类别内部的相对权重关系
2. **乘以一级指标权重**：确保类别间的比例（如安全性25%）
3. **全局验证**：确保所有20个二级指标权重之和=1

**示例**：
```
安全性类别（一级权重25%）：
  - 密钥泄露：修正后0.342 → 归一化后 0.0823 (占25%的32.9%)
  - 被侦察概率：修正后0.303 → 归一化后 0.0727 (占25%的29.1%)
  - 抗拦截能力：修正后0.395 → 归一化后 0.0951 (占25%的38.0%)
  
验证：0.0823 + 0.0727 + 0.0951 = 0.2501 ≈ 0.25 ✅
```

## 十、核心公式汇总

### 公式1：主观可信度
```
Ωi = 0.4 × δe + 0.6 × δk
```

### 公式2：变异系数（CV）
```
CV = (σ / μ) × 100%
```

### 公式3：专家离散度（专家CV）
```
专家CVi = (std(wij - μj) / μoverall) × 100%
```

### 公式4：客观可信度（分段衰减）
```python
if CV <= 15%:
    β = 1.0
elif CV <= 25%:
    β = 1.0 - (CV - 15) / 10 * 0.5
elif CV <= 35%:
    β = 0.5 - (CV - 25) / 10 * 0.5
else:
    β = 0.0
```

### 公式5：综合可信度
```
综合可信度 = 0.4 × (Ωi / 100) + 0.6 × βi
```

### 公式6（4-2）：专家最终可信度
```
λi = 0.5 × 综合可信度 + 0.5 × 平均把握度
```

### 公式7（4-15）：权重修正
```
ωj = Σ(λi × wij)
```

### 公式8：归一化
```
ωj_final = (ωj / Σ(ωk∈类别)) × 一级指标权重
```
